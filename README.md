# Flowwow — Цветочный магазин

## Запуск проекта

### 1. Запуск бэкенда (Spring Boot)
```bash
cd flower
./gradlew bootRun
```
Бэкенд запустится на http://localhost:8080

### 2. Запуск фронтенда (React + Vite)

##Установка зависимостей
```bash
cd flower-main
npm install
```
#Запуск

```bash
npm run dev
```
Фронтенд запустится на http://localhost:5173
