# Инструкция по резервному копированию 

## 1. Общая информация

| Параметр | Значение |
|----------|----------|
| База данных | PostgreSQL 15 |
| Имя БД | flowwow |
| Пользователь | postgres |
| Контейнер Docker | flowwow-postgres |
| Папка для backup | ./backups |
| Скрипт backup | backup.sh |
| Скрипт восстановления | restore.sh |

---

## 2. Скрипт backup.sh

### 2.1. Что делает скрипт

| Действие | Описание |
|----------|----------|
| 1 | Создает папку `./backups`, если её нет |
| 2 | Проверяет, запущен ли контейнер PostgreSQL |
| 3 | Создает backup базы данных |
| 4 | Сохраняет файл в формате `flowwow_backup_ГГГГММДД_ЧЧММСС.sql` |
| 5 | Удаляет файлы старше 30 дней |
| 6 | Показывает список всех backup |

### 2.2. Как запустить

```bash
# Перейти в корень проекта
cd /home/ksyunya/work/java/flowwow

# Запустить backup
./backup.sh
```
